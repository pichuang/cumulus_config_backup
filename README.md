CumulusLinux Config Export/Import
==========================

Intoduction
-----------

Export/Import configuration of CumulusLinux Switch by Ansible
- Timezone
- CumulusLinux Configuration
- Network Configuration
- User Auth File
- MOTD File

Requirement
-----------
- git
- ansible ( > 2.0.0)

Usage
-----
- For export:
```
bash run_export.sh
```

- For import:
```
bash run_import.sh
```
