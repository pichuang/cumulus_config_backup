CumulusLinux Config Export/Import
==========================

Intoduction
-----------

Export/Import configuration of CumulusLinux Switch by Ansible

Requirement
-----------
- git
- ansible ( > 2.0.0)

Usage
-----
- For export:
```
bash run_export.sh
```

- For import:
```
bash run_import.sh
```
